Guide for Learners

You can study Arka here.
Please work your way down the list on the left. Above all, you should read Lein's Lesson first.

If you want a beginners' course, then you should read Lein's Lesson.

If you want to understand what Arka is, then you should read Overview.

If you want a listening and speaking course, then you should read Listening.

If you want to understand Arka on a philosophical level, then you should read Studies, though it can be difficult to figure out its articles.

If you click Lein's Lesson 1, you will go to Greetings, not Introduction.

If you are unsure how to study Arka, then you should read How to Study (in Japanese).

If you are interested in conworlds and fantasies rather than conlangs, then you should read Fantasy Theory (in Japanese).

If you doubt that people can create a conlang in keeping with linguistics, then you should read How to create a conlang in keeping with linguistics (in Japanese).

If you doubt that Arka was made in keeping with linguistics, then you should read an Arka's consideration of cognitive linguistics (in Japanese).

If you want to know the history of the conworld, then you should read Ilmus (chronological table) (in Japanese).
